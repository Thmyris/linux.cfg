Setting Local Repository

The most perfect method for installing software package offline on Debian based GNU/Linux is : APT Local Repository.

Step-1 : Editing  /etc/apt/sources.list 
  • Open /etc/apt/sources.list  with gedit (or any other text editor) or with command-line as root
  • Add following entry at the top of file- /etc/apt/sources.list :
    • deb file:/var/local-deb-repo/ ./
  • Save file and exit from text-editor
Step-2 : Putting all debs inside  /var/local-deb-repo/
  • Create directory /var/local-deb-repo/
  • Put all deb files (including dependencies) inside this directory

Step-3 : Creating Packages index files

  • Run the following command:
    • cd /var/local-deb-repo
      sudo bash -c 'dpkg-scanpackages . | gzip > ./Packages.gz'

Step-4 : Updating APT package source

  • Run following command:
    • sudo apt-get update
Now you can install packages by apt using sudo apt-get install <package> command.
Convenience/Advantage/Flexibility of this method:-
  • If you want to install another package, then you have to follow only Step 2 to 4.
  • In other word, Just maintain appropriate debs inside /var/local-deb-repo/  and you can directly go for couple of commands (Step 3 & 4).
  • If you want to clean local repository, then just empty /var/local-deb-repo/ and repeat Step 3 & 4!
